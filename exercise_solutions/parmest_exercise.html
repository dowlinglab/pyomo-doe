Exercise: ParmEst#

In this exercise, you will practice using ParmEst to estimate four parameters (\(U_a\), \(U_b\), \(C_p^H\), \(C_p^S\)) in the TCLab model using the step test data. We previously estimated these parameters using the sine test data.

Parameter estimation with ParmEst#

Now for the main event: performing nonlinear least squares with ParmEst.

# Call ParmEst
### BEGIN SOLUTION
import pyomo.contrib.parmest.parmest as parmest

parmest_create_model = lambda data: create_model(
    data=data, mode='parmest', number_of_states=2
)

# define the parameter names as a list
# two-state model
theta_names = ['Ua', 'Ub', 'inv_CpH', 'inv_CpS']

# four-state model
# theta_names = ['Ua', 'Ub', 'Uc','inv_CpH', 'inv_CpS']

# Create a parameter estimation object
pest = parmest.Estimator(
    parmest_create_model,
    data=[
        tc_data.to_data_frame()
    ],  # Needs to be a list of dataframes, otherwise parmest threats each row as a separate model
    theta_names=theta_names,
    tee=True,
)

obj, theta = pest.theta_est()

obj, theta, cov = pest.theta_est(calc_cov=True, # boolean to calculate the covariance matrix
                                    cov_n = len(tc_data.time) # number of data points in the least squares objective
)
